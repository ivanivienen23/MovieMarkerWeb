# MyMovies

MyMovies es una aplicación web en React que permite explorar películas populares, buscarlas por nombre y gestionar tus películas favoritas. La app utiliza la API de [The Movie Database (TMDB)](https://www.themoviedb.org/) para obtener información actualizada sobre películas.

---

## Características

- Visualización de películas populares al cargar la página.
- Búsqueda de películas por título.
- Guardado de películas favoritas en `localStorage`.
- Navegación entre Home y Favorites con React Router.
- Tarjetas de películas con imagen, título, año y overlay de favoritos.
- Navbar responsivo y siempre visible.

---

## Tecnologías

- React
- React Router
- Context API (para gestión global de favoritos)
- CSS global (sin Tailwind obligatorio)
- TMDB API
